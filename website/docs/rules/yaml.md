### Boolean values

![yaml](https://img.shields.io/badge/yaml-truthy-cb171e)

Only `true` and `false` are allowed as boolean values.

**:material-star-four-points-outline:{ #accent } Before**

```yaml
foo: on
bar: OFF
```

**:material-star-four-points:{ #accent } After**

```yaml
foo: true
bar: false
```
