    What is Stylebook?


    • Library extension
      A set of additional rules for static code analysis tools

    • Simplified CLI
      One command to lint multiple file types

    How does it work?

    flowchart LR
  subgraph "Project"
    src(Source code)
    config1[Optional config files]
  end
  subgraph "Static analysis"
    linters[Linter libraries]
    extensions[Rule extensions]
    config2[Default config files]
  end
  subgraph "Report"
    cli[Build failure<br>on lint errors]
  end
  subgraph "Refactor"
    manual(Manual edits<br>using suggestions)
  end

  src -- run --> linters
  extensions -- CLI --> cli
  cli -- fix --> manual

    Support list¶

    Many thanks to the communities who maintain the linter tools!

    One CLI to rule them all

    • Blinter

      Blinter is a linter for Windows batch files.

      PyPI

    • checkmake

      Linter/analyzer for Makefiles.

    • ChkTeX

      LaTeX semantic checker.

      Website

    • Stylelint

      A mighty CSS linter that helps you avoid errors and enforce conventions.

      NPM   Website

    • csvlint

      Library and command line tool that validates a CSV file.

    • dotenv-linter

      ☺️ Linting dotenv files like a charm!

      PyPI   Website

    • gomoddirectives

      A linter that handle directives into go.mod.

    • hadolint

      Dockerfile linter, validate inline bash, written in Haskell.

    • HTMLHint

      ⚙️ The static code analysis tool you need for your HTML.

      NPM   Website

    • JSON Lint

      JSON/CJSON/JSON5 parser, syntax & schema validator and pretty-printer with a command-line client, written in pure JavaScript.

      NPM   Website

    • lockfile-lint

      Lint an npm or yarn lockfile to analyze and detect security issues.

      NPM

    • libxml2

      A Pluggable Terraform Linter.

    • Maid

      A lightweight Mermaid diagram linter which helps humans and AI with perfect diagrams.

      NPM   Website

    • markdownlint

      A Node.js style checker and lint tool for Markdown/CommonMark files.

      NPM

    • propertieslint

      A linter for .properties file.

    • protolint

      A pluggable linter and fixer to enforce Protocol Buffer style and conventions.

    • py-taplo

      Python wrapper for taplo TOML toolkit.

      PyPI   Website

    • pyinilint

      A linter for INI-like configuration files which are intended to be parsed by the standard Python configparser module.

      PyPI

    • ShellCheck

      ShellCheck, a static analysis tool for shell scripts.

      Website

    • SQLFluff

      A modular SQL linter and auto-formatter with support for multiple dialects and templated code.

      PyPI   Website

    • TFLint

      A Pluggable Terraform Linter.

    • yamllint

      A linter for YAML files.

      PyPI
